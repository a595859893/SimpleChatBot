# SimpleChatBot
Version 6

允许链式推导，多条件推导多条件，tag引用
 

指令
```
教导实体：进入教导实体模式，从而告诉机器人某些词语属于什么类型
教导联系：进入教导联系模式，从而告诉机器人什么实体能推导出什么实体
教导对话：进入教导句子模式，从而教会机器人句子
聊天：进入聊天模式，让机器人回应你说的话，如果没有找到，则当场进行询问
```

```
你      教导实体
机器人  进入实体学习模式
你      人名:小明;水果:苹果 香蕉 梨子;颜色:红 黄;形状:圆 弯
机器人  我明白了！
你      教导联系
机器人  进入联系学习模式
你      水果:苹果<->颜色:红 形状:圆
机器人  我明白了！
你      水果:梨子<->颜色:黄 形状:圆
机器人  我明白了！
你      水果:香蕉<->颜色:黄 形状:弯
机器人  我明白了！
你      人名:小明 颜色:红 水果->喜好:喜欢
机器人  我明白了！
你      人名:小明 形状:弯 水果->喜好:讨厌
机器人  我明白了！
你      教导对话
机器人  进入对话学习模式
你      [人名]喜欢[水果]吗?
机器人  我应该怎么回复？
你      [?喜好 "喜欢"][人名]觉得很棒;[?喜好 "讨厌"]最好不要给[人名][水果];[!喜好]我不太确定
机器人  我明白了！
你      [水果|1]和[水果|2]形状一样吗?
机器人  我应该怎么回复？
你      [?形状{水果|1} 形状{水果|2}]我觉得一样;[!形状{水果|1} 形状{水果|2}]应该是不一样的
机器人  我明白了！
你      聊天
机器人  进入聊天模式
你      小明喜欢苹果吗?
机器人  小明觉得很棒
你      小明喜欢梨子吗?
机器人  我不太确定
你      小明喜欢香蕉吗?
机器人  最好不要给小明香蕉
你      苹果和梨子形状一样吗?
机器人  我觉得一样
你      梨子和香蕉形状一样吗?
机器人  应该是不一样的
```
